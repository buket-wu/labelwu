# `LB-Components`

基于 LB-Annotation 开发，即插即用的标注组件库。支持拉框、标签、多边形、线条、点、文本标注工具。开发者可根据需求配置，快速实现多种标注功能。

## 📦 Install

```bash
# NPM
$ npm install @labelwu/lb-components

# YARN
$ yarn add @labelwu/lb-components
```

## 🪵 Requirements

| PureDependencies | Version  |
| ---------------- | -------- |
| react            | >=16.9.0 |
| antd             | >=4.15.0 |
